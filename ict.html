üíª Computer Systems

A computer system consists of hardware and software components that work together to perform tasks.

Components:

  • Hardware: CPU, RAM, motherboard, etc.
  • Software: Operating system, applications.

Root Concept:

First designed by **Charles Babbage** in the 19th century.

Imitative Explanation:

A computer is like a **human body**: - CPU = **Brain** - RAM = **Short-term memory** - Storage = **Long-term memory**

üåê Networks

A computer network is a group of devices that communicate to share resources.

Types of Networks:

  • LAN: Small area (e.g., office).
  • WAN: Large distances (e.g., the internet).

Root Concept:

Started with **ARPANET** in the 1960s.

Imitative Explanation:

A network is like a **road system**, where: - Data packets = **Cars** - Routers = **Traffic signals**

üñ•Ô∏è Software

Software refers to the programs that allow computers to function.

Types of Software:

  • System Software: Operating systems.
  • Application Software: Programs like Word, Photoshop.

Root Concept:

First software was written by **Ada Lovelace** in the 1840s.

Imitative Explanation:

Software is like a **restaurant**: - OS = **Kitchen** - Apps = **Chefs**
